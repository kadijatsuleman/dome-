This is a simple front-end project for a church website, including:

- **Home Page**:  
  - Hero section with welcome message  
  - About section with church information  
  - Gallery of images (choir, groups)  

- **Sermon Page**:  
  - Hero section with title  
  - List of recent sermons with titles, dates, and descriptions  
  - Two sermon images  

## Technologies Used

- HTML5  
- CSS3 (responsive design)  

## Features

- Fully responsive layout for desktop, tablet, and mobile  
- Hero sections with centered heading and paragraph  
- Clean navigation bar linking Home and Sermon pages  
- Footer styled with Dunamis color theme
